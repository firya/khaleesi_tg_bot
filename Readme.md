## To set webhookurl
`https://api.telegram.org/bot{BOT_TOKEN}/setWebhook?url=https://{BOT_DOMAIN}`

## To get webhookurl
`https://api.telegram.org/bot{BOT_TOKEN}/getWebhookInfo`
